=== Jenkins-Pipeline

[plantuml,jenkins,png]
----
@startuml
rectangle gitrepo
queue jenkins{
    file java
    storage compile
    file class
    storage package
    file jar
    storage test
    storage deploy
}
gitrepo -> java
java -> compile
compile -> class
class -> package
package -> jar
jar -> test
test -> deploy
@enduml
----
